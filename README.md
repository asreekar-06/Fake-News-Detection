# Fake News Detection System

## 📌 Overview
This project is a machine learning-based application that detects whether a news article is real or fake using NLP techniques.

## 🚀 Features
- Input news text
- Text preprocessing
- Classification using ML model
- Predicts whether news is Real or Fake

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Pandas
- NLP techniques

## ⚙️ How It Works
1. Input news text
2. Clean and preprocess text
3. Convert text to numerical features
4. Apply trained model
5. Output prediction (Real/Fake)

## 📊 Sample Output
Input: "Breaking news about AI..."  
Output: Fake News
